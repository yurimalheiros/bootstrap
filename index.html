Frameworks

O que é um framework front-end?

Uma coleção de componentes HTML, CSS e JavaScript prontos para usar nas nossas páginas web.

O que é um framework front-end?

Frameworks CSS - Geralmente incluem um reset, um sistema de grid e estilos para tipografia e formulários. Alguns incluem suporte a dispositivos móveis e para impressão.

Frameworks JavaScript - Fornecem estrutura para o código JavaScript e o mantém organizado.

O que é um framework front-end?

Responsividade e consistência entre navegadores.

Componentes modulares e reusáveis.

O que é o Bootstrap?

Bootstrap é um framework front-end focado na interface do usuário.

Aumenta a velocidade de desenvolvimento de novos websites e aplicações.

Um dos projetos mais populares do GitHub com ~100.000 estrelas e mais de 44.000 forks.

O Bootstrap é um projeto de código aberto no GitHub, então qualquer um pode contribuir com seu progresso.

O que vem com o Bootstrap?

Você pode fazer o download de todos os arquivos necessários em getbootstrap.com ou no repositório do GitHub.

Obs: Vamos usar o Bootstrap 4, mas a versão 3 ainda é bastante popular.

Estrutura dos Arquivos

Layout:

Opções para estruturar sua página com o Bootstrap, incluindo estilos globais, sistema de grid e uma base de apoio para iniciar o desenvolvimento.

Conteúdo:

Estilos para exibir conteúdo com os elementos mais comuns do HTML, incluindo tipografia, imagens, tabelas, etc.

Componentes:

Mais de uma dúzia de componentes reusáveis como botões, dropdowns, campos de entrada, alertas, entre outros.

Sistema de Grid

O Bootstrap é baseado num sistema de grid com 12 colunas. Ele usa classes CSS para decidir o tamanho de cada elemento.

Cada elemento pode ocupar o espaço de 1 até 12 colunas.

Algumas Regras:

  • Colunas devem ser colocadas dentro de uma classe .container (tamanho fixo) ou .container-fluid (tamanho fluido) para o alinhamento e padding ficarem corretos.
  • Use linhas (.row) para criar um agrupamento horizontal de colunas.
  • O conteúdo deve ser colocado em colunas e apenas colunas devem ser filhas das linhas.

Tipografia

As classes básicas da tipografia normalizam cabeçalhos, fontes dos parágrafos, estilo de lista, entre outros.

Classes utilitárias de texto também são incluídas.

Tipografia

Imagens Responsivas

Imagens no Bootstrap 4 podem ser definidas como responsivas adicionando a classe .img-fluid.

Ela aplica max-width: 100%; e height: auto; para que a imagem redimensione ficando do tamanho do elemento pai.

Outros Estilos de Imagem

  • .img-rounded arredonda as bordas da imagem
  • .img-circle aplica o estilo border-radius: 50%;
  • .img-thumbnail adiciona uma borda extra arredondada

Formulários, Modais e Alertas

Formulários

Adicionar estilo a formulários é rápido e fácil usando o Bootstrap.

.form-group

Use a classe .form-group para organizar seções do formulário.

Pode ser usado em elementos de blocos como fieldset ou div.

.form-group

.form-control

Usado para dar estilo a...

  • Inputs de texto (e.g. text, number, email)
  • Selects
  • Textareas

.form-control

.radio e .checkbox

Usado para dar estilo para radios e checkboxes.

.radio e .checkbox

Botões

Adicione uma classe .btn junto com uma classe de estilo.

O Bootstrap tem seis estilos e variações.

Botões

Botões

Use .btn-lg ou .btn-sm para botões maiores ou menos respectivamente.

Use .btn-block para botões de bloco.

Use o atributo disabled para desabilitar o botão.

Modais

Versões mais bonitas para dialogs!

Defina um atributo data-target com o valor do ID do modal.

Consiste em um header, body, e footer.

Modais

Alertas

Use .alert para prover mensagens de feedback para o usuário.

Bootstrap tem quatro estilos: alert-*.
